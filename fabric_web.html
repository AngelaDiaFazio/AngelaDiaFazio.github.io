Microsoft Office UI Fabric

Documentation Site

Fabric is an open source Front End UI toolkit for building experiences in Office and Office 365

The problem was that there was not a good reference, or documentation site for users.

The solution was to create an easy to reference guide online.

he information was organized into: Getting started, Styles, Components, Samples, and Resources.

The styles & components page were approached to allow for a clear understanding of what is available both visually and in code.

A right nav was introduced on the components page as a way to navigate the different variants of each control. IE: Standard and Primary Buttons
