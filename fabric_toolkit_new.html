Microsoft Fabric

2015 - 2016

Fabric is an open source Front End UI toolkit for building experiences in Office and Office 365. The framework team needed a set of centralized assets that contributed to the stealth and efficiency for the Fabric design studio.

Contents

The toolkit consists of several assets including: UX Flow Templates, Adobe CC Libraries, Component Templates, Fabric Documentation.

UX Flow Templates

To help frame responsive designs

Component Templates

These templates were designed to help designers submit their design to the framework.

Adobe CC Libraries

After creating the component assets, they were placed in a several CC libraries

Components

Before I began the project, an inventory of the o365 grid system & existing components were taken. This is a dynamic system with many complex pieces. My job was to analyze& scrub the files. Each file was perfected and placed into the library. These are tools designers can use as guidance while working on designs.
