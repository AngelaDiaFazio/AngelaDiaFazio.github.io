Living Style Guide

CSS Styling for Front End Framework

The approach for this Front End Style Guide was based on Brad Frost's Atomic Design. This breaks down each component into "atoms". The atoms can then be used to build "molecules". For example, a button, font, and input field can be used to create a submission form. The guide is generic enough to be used by any brand, and is effortlessly scalable.

Sass was used with partials to organize the file structure.

Built in Node, Each atom is represented with it's own view/route.
